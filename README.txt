MILANO RACING WEBSITE

Open index.html in a browser to view the site.

Files:
- index.html: page structure and content
- style.css: styling, responsive design, and animations
- script.js: navigation, reveal animations, tabs, counters, and form interaction
- assets/images/logo.svg: temporary editable logo added because the original logo.jpg was missing from the uploaded project

To use your real logo:
1. Put your logo in assets/images/
2. Either name it logo.svg, or update the three logo paths in index.html

Note: The contact form currently demonstrates the interaction only; it does not send email to a server.
